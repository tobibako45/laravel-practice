# Usage

Laravel Framework 5.5.45

```
composer create-project --prefer-dist laravel/laravel PROJECT_NAME "5.5.*"
```

```
$ docker-compose build
```

docker起動
```
$ docker-compose up -d
```

phpコンテナに入ります
```
$ docker-compose exec php bash
```

Laravelのインストール
```
$ laravel new
```

Laravel ヴァージョン確認
```
$ php artisan -v
```

You can access localhost:8000
